## para levantar el proyecto

    1- iniciar docker desktop
    2- tirar comando docker compose up -d
    3-  correr migraciones npx prisma migrate dev
    4-Ejecutar seed `npm run seed`
    5-Correr el proyecto ` npm run dev`
